## Active Directory Home Lab

# Project Overview
This project demostrates the deployment and administration of a Windows Server 2022 Active Directory environment inside VirtualBox.

The lab simulates a small enterprise network with departmental users, security groups, role-based access controls, shared resources, and Group Policy management.

# Objectives

- Install Windows Server 2022
- Configure Active Directory Domain Services
- Create a new domain
- Create Organizational Units (OUs)
- Configure users and groups
- Implement Role-Based Access Control (RBAC)
- Configure Group Policy Objects (GPOs)
- Join a Windows 11 client to the domain
- Validate user authentication and authorization

## Technologies Used

- VirtualBox
- Windows Server 2022
- Windows 11 Enterprise
- Active Directory Domain Services
- DNS
- Group Policy Management
